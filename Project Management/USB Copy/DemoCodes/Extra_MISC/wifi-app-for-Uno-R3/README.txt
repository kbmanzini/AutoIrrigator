If you intend to use the Arduino UNO R3 Mini or the standard Arduino UNO R3 with this kit, you can enhance your setup by replacing the existing libraries. To do this, simply copy the provided 'libraries' folder to your Arduino installation directory, replacing the current one. This action will introduce two new libraries essential for your project:

1. **GeeekPiMQTT** - A library designed for MQTT communication.
2. **WiFiEsp** - A library facilitating the use of ESP8266 WiFi modules.

The primary demonstration code provided is **MQTT_ESP01.cpp**. Please note that this is a sample code and may not function optimally with the Arduino UNO R4 WiFi. It serves as a starting point for those who wish to experiment.

We offer these libraries for your exploration and learning purposes. However, please be aware that we do not provide technical support for this specific setup. It is intended for those who are eager to explore and learn on their own.

Thank you! have a nice day !
